简介

在 C++ 中，extern关键字主要用于声明变量或函数是在其他文件中定义的。它的作用是告诉编译器，这个变量或者函数的定义在别的地方，先不要为这个变量或函数分配内存空间，等到链接的时候再去寻找它的定义。

如果你有一个大型项目，包含多个源文件（.cpp文件），可能在一个文件中定义了一个全局变量或者函数，而在另一个文件中需要使用它。这时候就可以使用extern关键字来进行声明。

函数声明中的extern

对于函数，extern声明也是类似的。在默认情况下，C++ 函数声明是extern的，这意味着函数可以在其他文件中定义。

extern “C”的特殊情况

当 C++ 代码需要调用 C 函数或者 C 库时，就需要使用extern "C"。这是因为 C++ 的函数名在编译时会进行名字修饰，而 C 函数不会。

extern关键字在 C++ 中对于跨文件的变量和函数使用以及 C/C++ 混合编程等场景中起着非常重要的作用，它能够帮助编译器正确地处理程序的链接过程。
